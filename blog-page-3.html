HTML and CSS

Jamodin Maliwanag - October 3, 2021

Have you ever wondered how websites are made? HTML and CSS are coding languages that are used to make websites. Both HTML and CSS help users navigate through the internet with ease

What makes up HTML?

HTML is an abbreviation for HyperText Markup Language. The “HyperText” is what links websites together through clicking on text or links. The “Markup” is how pages are shown by use of images or sections of text. HTML is made up of elements, tags and attributes.


An HTML element includes the HTML tag and the content inside. An HTML tag is used to group or transform text to do almost anything. A “section tag” can group text together, and that text can be inside a “p tag” which has a default font size and font weight. HTML tags can include HTML attributes to provide additional information. What makes up CSS?

CSS is Cascading Style Sheets and it helps style and design the look of websites. CSS has to be linked to HTML for it to work. Without CSS, a website is just plain text and images on a white background. CSS is made of syntax rule, selector, property and value.


The CSS rule contains the CSS selector and CSS value. The CSS selector is used to identify the HTML tag, like the “p tag”, and target it. The CSS value and property is what you want to do to the content inside the tag. An Example is of what it looks like is:

The p tag is targeted, and the text inside of it is changed to a font size of 14 pixels and the text color is changed to blue.


HTML and CSS can be used to make any websites look any way you want. Coding for these languages is easily accessible since all you need is Notepad, which comes pre-installed on your computer. However, VSCode is a preferred program since it provides many tools and shortcuts to help with coding faster and easier.
